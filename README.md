## Enviroment
 - Python3
 - QGIS 3.14
 - sfo_nodes.txt

## Usage
* run `python NetworkAnalysis.py --start_node  "输入节点" --end_node  "输出节点"  --path_output "最短路径" --status_output "状态文件"`  
In the path_output file, you will get the shortest path computed by dijkstra algorithm.
